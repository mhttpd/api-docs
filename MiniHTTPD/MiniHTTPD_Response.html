Class MiniHTTPD_Response

Description
Description | Vars (details) | Methods (details)

The MiniHTTPD response class.

Apart from overloading any members in the base message class, the main responsibility of this class is to build the response for returning to the client, and in particular to combine the server client response with any FCGI client response. The response body may be buffered locally or handled as a stream resource. Most of the methods here are also chainable.
