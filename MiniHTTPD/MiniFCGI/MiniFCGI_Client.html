Class MiniFCGI_Client

Description
Description | Vars (details) | Methods (details)

The MiniFCI client class.

This class is used to communicate with a running FastCGI process on behalf of a server client. It handles the connections, parses and sends the FCGI request, and applies the FCGI protocol through the record objects that it creates. The responses are passed back to the server client typically via a bound object.
