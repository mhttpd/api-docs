Class MiniHTTPD_Handler_Rewrite

Description
Description | Vars (details) | Methods (details)

This handler rewrites request details based on regex matches of URL strings.

To add rewrite rules: extend this class and define the rules in $rules, then load the extended handler via the classes.php and ini files. A simpler method is to define the rules in a custom file and load it via the main server ini file (see the loadRules() method for more details).

The 'default' rule given as an example allows the use of pretty URLs as used by various web frameworks such as Kohana, CI, etc.
