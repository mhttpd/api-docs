Class MiniHTTPD_Server

Description
Description | Vars (details) | Methods (details) Constants (details)

The main MiniHTTPD server class.

This static class deals with the most basic level of the server's operation. It initializes the main environment, sets up the listening socket, handles all client connections, and monitors any FastCGI connections for processing by the created client objects. It also includes a number of useful helper methods, e.g. an object factory, access to the configuration settings, etc.

		static main (line 783)
	
 
	
	

Runs the main server loop.


This is where the server does most if its work. Once the listening socket  is established, iterations of the the main loop are controlled entirely by  stream_select() for both client connections and FCGI requests as well as any  open file streams. Each loop should ideally finish as quickly as possible to  enable best concurrency.
