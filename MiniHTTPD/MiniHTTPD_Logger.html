Class MiniHTTPD_Logger

Description
Description | Vars (details) | Methods (details)

The MiniHTTPD logger class.

This class provides a simple framework for logging the server's activities. By default it provides functionality for an access log in Apache style format, although this is easy to customize. Log entries may be buffered for batch writing to disk, depending on the configuration.
