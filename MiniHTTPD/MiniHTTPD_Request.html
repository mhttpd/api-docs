Class MiniHTTPD_Request

Description
Description | Vars (details) | Methods (details)

The MiniHTTPD request class.

Apart from overloading any members in the base message class, the main responsibility of this class is to provide the information needed by the client to process the request and determine the relevant response. Much of this class therefore consists of accessor or query methods.

rewriteUrlPath (line 221)

Rewrites the path element of the request URL to allow transparent virtual to real path mapping.

This is especially useful for fixing issues with links given as relative paths where the browser will automatically add the base path. Any virtual path elements (i.e. ones that can't be found in the docroot) should be rewritten in this case usually to just '/'.

  • return: this instance
  • access: public

  • string $search: the regex search pattern
  • string $replace: the replacement value
