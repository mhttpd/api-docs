Class MiniHTTPD_Request

Description
Description | Vars (details) | Methods (details)

The MiniHTTPD request class.

Apart from overloading any members in the base message class, the main responsibility of this class is to provide the information needed by the client to process the request and determine the relevant response. Much of this class therefore consists of accessor or query methods.
