Loads the required class files and sets the class aliases.

To extend any of the classes transparently, first copy this file and rename it to 'user_classes.php'. Edit the new file to load your extended class definitions immediately after the original ones, then change the relevant class aliases to point to your extended classes. For example:

  1.    require 'classes\logger.php';
  2.    require 'myclasses\mylogger.php';
  3.    // class My_Logger extends MiniHTTPD_Logger {}
  4.    class_alias('My_Logger', 'MHTTPD_Logger');

All of the other classes will now be able to make use of this extended class without any of the scripts needing to be edited, since they only reference the class aliases directly, not the original class names.
